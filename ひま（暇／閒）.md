# ひま【暇／▽閒】

**\#1**［名］
1. 継続する動作などの合間に生じるわずかの時間。「食事をする─もない」「─を見て外出する」
2. 事をするための一定の時間。「─のかかる仕事」「手間─」
3. 自由に使える時間。なすべきことの何もない時間。「─を持てあます」
4. 休暇。休み。「一週間の─をいただく」
5. 主従・夫婦などの関係を斷つこと。縁を切ること。
    

**\#2**［形動］［文］［ナリ］自由になる時間がふんだんにあるさま。なすべきことの何もないさま。「明日は─だ」「商売が─になる」「─な人」
