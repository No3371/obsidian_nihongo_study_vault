# のぼ・せる【逆＝上せる】

［動サ下一］［文］のぼ・す［サ下二］
1. 血が頭へのぼって、ぼうっとなる。上気する。「長湯で─・せる」「暑さで─・せる」
2. 興奮して理性を失う。血迷う。逆上する。「─・せて前後の見境がなくなる」
3. すっかり夢中になる。熱中する。「アイドル歌手に─・せる」
4. 思い上がる。うぬぼれる。「成功したからといって─・せるな」
