# とど・く【屆く】
**\#1**［動カ五（四）］

1.  ある所にまで至りつく。達する。及ぶ。「四十に手が─・く」「遠くまで─・く聲」
2.  送った品物や郵便物が相手の所に著く。「母から便りが─・く」「贈り物が─・く」
3.  注意などが十分に行きわたる。行き屆く。「親の目が─・く」「細かいところまで神経が─・く」
4.  願い事がかなう。気持ちが通じる。「祈りが─・く」「誠意が─・く」
