# まったく【全く】

［副］《形容詞「まったい」の連用形から》
1.  完全にその状態になっているさま。すっかり。「全く新しい企画」「回復の希望は全く絶たれた」
2.  打消しの語を伴って、完全な否定の意を表す。決して。全然。「彼は事件とは全く関係がない」「全く話にならない」
3.  ある事実・判断を強調する気持ちを表す。本当に。実に。「今日は全く寒い」「全くけしからん話だ」「全く君の言う通りだよ」
4.  （「まったくだ」「まったくです」などの形で）話し言葉で、相手の言葉を受けて、それを強く肯定または否定する意を表す。「『腹が立つといったら、殴りたい気分だよ』『全くだね』（肯定）」「『旅行の準備はできたかね』『全くですよ』（否定）」
5.  （「まったくの」の形で）話題になっていることについて、その通りの、文字通りの、嘘偽りなしなどの意を表す。「抜けているといったら、全くの抜け作さ」「料理については全くの素人だ」
6.  （感動詞的に用いて）強い怒りや失望などの気持ちを表す。「全線不通とは、全く」
