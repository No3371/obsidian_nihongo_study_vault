# はた・す【果（た）す】

［動サ五（四）］
1.  物事を[成し遂げる](なしとげる（成し遂げる）)。「約束を―・す」「望みを―・す」
2.  その立場としての仕事を[みごと](みごと（見事／美事）)にやってのける。「指導的役割を―・す」
3.  命を終わらせる。殺す。「―・さずにおくものか」
4.  （動詞の連用形に付いて）すっかり…する。…してしまう。「金を使い―・す」
    

\[可能\]**はたせる**
