# ま・げる【曲げる】

［動ガ下一］［文］ま・ぐ［ガ下二］
1. まっすぐな物などを弓形にする。まがった状態にする。たわめる。「針金を―・げる」「腰を―・げる」
2. 傾ける。傾斜させる。「首を―・げる」
3. （「枉げる」とも書く）本来の主義・考え・希望などをむりに変える。「信念を―・げる」
4. （「枉げる」とも書く）道理・事実などを意図的にゆがめる。「法を―・げる」「事実を―・げた報道」
5. 《「質」と同音の「七」の字の第2画を曲げるところから》質に入れる。「時計を―・げる」
