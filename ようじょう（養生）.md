# よう‐じょう〔ヤウジヤウ〕【養生】

［名］(スル)
1. 生活に留意して健康の増進を図ること。摂生 (せっせい) 。「酒やタバコをひかえ、つね日頃から養生している」
2. 病気の回覆につとめること。保養。「転地して養生する」
3. 打ち込んだコンクリートやモルタルが十分に硬化するように、低溫・乾燥・衝撃などから保護する作業。
4. 家具の運搬や塗裝作業などの際に、運搬物や周囲の汚損を防ぐために布や板などで保護すること。「養生シート」
