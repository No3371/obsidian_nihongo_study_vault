# やかまし・い【×喧しい】

［形］［文］やかま・し［シク］
1.  声や物音などが騒がしい。うるさく、不快である。「工事現場の―・い音」
2.  いろいろの人が話題にして騒がしい。また、評判が高い。「世間が―・い」
3.  こまごまとしていて、めんどうくさい。また、小言が多くてわずらわしい。「何事にも―・い人」「書類の手続きが―・い」
4.  きびしい。「しつけに―・い家庭」「規則が―・い」
5.  好みなどが気むずかしい。「食べ物に―・い人」
