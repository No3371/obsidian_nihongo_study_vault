# なら・す【鳴らす】

［動サ五（四）］
1.  音を出すようにする。音をひびかせる。「鐘を―・す」「ブザーを―・す」「指を―・す」
2.  そのことで大変世間の評判をとる。「昔は名投手で―・したものだ」
3.  やかましく言いたてる。「不平を―・す」「非を―・す」
4.  音を出して放屁する。おならをする。    
    1.  「誤りていと高く―・してけり」〈[今昔](/word/%E4%BB%8A%E6%98%94%E7%89%A9%E8%AA%9E%E9%9B%86/#jn-83551)・二八・一〇〉
