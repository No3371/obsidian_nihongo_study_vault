# 手(て)を付(つ)・ける

1.  着手する。仕事などを始める。「行政改革に―・ける」
2.   使い始める。また特に、使い込む。「定期預金に―・ける」「公金に―・ける」
3.   料理などを消費する。「刺身に―・ける」
4.   目下の女性などと関係をもつ。
