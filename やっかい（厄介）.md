# やっ‐かい〔ヤク‐〕【厄介】

［名・形動］

1. めんどうなこと。扱いに手数がかかり、わずらわしいこと。また、そのさま。「厄介なことに巻き込まれる」
2. めんどうをみること。また、世話になること。「親の厄介になる」
3. 他家に寄食すること。居候 (いそうろう) 。
    →[面倒 (めんどう)](https://dictionary.goo.ne.jp/word/%E9%9D%A2%E5%80%92/#jn-218255)［用法］
