# あやまち【過ち／▽誤ち】

1. まちがい。失敗。「―を恐れず試みる」
2. 犯してしまった罪。[過失](https://dictionary.goo.ne.jp/word/%E9%81%8E%E5%A4%B1/#jn-40528)。「―を償う」
3. 男女間の不倫。不義。
4. けが。負傷。
    >「―すな。心して降 (お) りよ」〈[徒然](https://dictionary.goo.ne.jp/word/%E5%BE%92%E7%84%B6%E8%8D%89/#jn-148773)・一〇九〉
