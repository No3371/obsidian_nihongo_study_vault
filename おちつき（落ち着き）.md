# おち‐つき【落（ち）著き／落（ち）付き】

1. 気持ちや態度が物事に動じないで、安定した狀態にあること。また、そのような狀態・態度。平靜さ。「─がない子供」「─を取り戻す」
2. 動いていた物事が安定した狀態になること。「相場の─を待つ」「世の中が─を失う」
3. 器物の座りぐあい。また、物事の調和のぐあい。「─の悪い置物」「下 (しも) の句の─が悪い」
4. 旅館などに著いたとき、最初に飲食するもの。    
    1.  「─は御雑煮、餅 (もち) は大方一人前」〈浄・傾城酒呑童子〉
