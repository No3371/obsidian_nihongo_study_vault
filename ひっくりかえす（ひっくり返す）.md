# ひっくり‐かえ・す〔‐かへす〕【引っ繰り返す】
［動サ五（四）］《「ひっくりがえす」とも》

1.  上下・表裏などを反対にする。さかさまにする。裏返す。「名札 (なふだ) を―・す」
2.  書物や文書などのページを繰る。「ノートを―・して調べる」
3.  横や後ろに勢いよく倒す。転倒させる。「つまずいてバケツを―・す」
4.  今までと反対の状態・関係にする。くつがえす。逆転させる。「会議の決定を―・す」「定説が―・される」
