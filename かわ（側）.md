# かわ〔かは〕【▽側】

1. 物の一つの方向・方面。一面。「家の西の側」「左っ側の目が痛い」
2. 相対するものの一方。「労働者の側に立つ」
3. 物のまわりを取り囲んだり覆ったりしているもの。「時計の側」
4. 列。並び。「右寄りの二側に女子が座る」→[がわ（側）](https://dictionary.goo.ne.jp/word/%E5%81%B4_%28%E3%81%8C%E3%82%8F%29/#jn-46829)
