# さかん【盛ん】

［形動］［文］［ナリ］《「さかり」の音変化》
1.  勢いがいいさま。「盛んな声援がとぶ」「盛んに燃える」「食欲が盛んだ」
2.  人が最も元気な時期にあるさま。気力などが充実しているさま。「人生の盛んな時代」「老いてますま盛んだ」「血気盛ん」
3.  盛大に行われるさま。繁盛しているさま。また、広く行われるさま。「スポーツの盛んな学校」「盛んな歓迎会」「貿易が盛んになる」「欧米ではサッカーが盛んだ」
4.  積極的に繰り返し行われるさま。熱心。「学者の間で盛んな議論が交わされた」
