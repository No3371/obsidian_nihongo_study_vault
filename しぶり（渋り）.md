# しぶ・い【渋い】

［形］［文］しぶ・し［ク］

1. 渋柿を食べたときなどの、舌がしびれるような味である。「―・いお茶」
2. はででなく落ち着いた趣がある。じみであるが味わい深い。「―・い声」「―・い色のネクタイ」「目の付け所が―・い」
3. 不愉快そうな、または、不満そうなようすである。「―・い顔をする」
4. 金品を出すのを嫌がるようである。けちである。「―・い客」
5. 動きが滑らかでない。「湿気でふすまが―・くなる」
    

\[派生\]**しぶさ**［名］
