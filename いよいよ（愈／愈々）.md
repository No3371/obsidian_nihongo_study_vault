# いよ‐いよ【×愈／×愈々】

［副］

1. 持続的に程度が高まるさま。ますます。より[一層](いっそう（一層）)。「―雨が激しくなる」
2. 不確定なものが確定的になるさま。まさしく。「昇進は―明らかだ」「―もって彼が怪しい」
3. 待望していた物事が成立したり実現したりするさま。とうとう。ついに。「新社屋が―完成する」「―試合開始だ」
4. 重大な事態に至ろうとしているさま。いざ。「―の時は加勢を頼むよ」
