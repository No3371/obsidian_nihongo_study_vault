# つなが・る【×繋がる】

［動ラ五（四）］
1.  離れているものが結ばれて、ひと続きになる。「島と島とが橋で―・がる」「電話が―・る」「光回線が―・る」
2.  つらなり続く。また、継続する。「車が一〇キロも―・っている」「この道は国道に―・っている」「首が―・る」
3.  関係がある。[結びつく](むすびつく（結び付く）)。「事件に―・る遺留品」「努力が成功に―・る」
4.  血筋が同じである。血縁関係がある。「血の―・った人」
5.  ひかされる。ほだされる。「情に―・る」
