# かぞ・える〔かぞへる〕【数える】

［動ア下一］［文］かぞ・ふ［ハ下二］

1. 数量や順番を調べる。勘定する。「人数を―・える」「指折り―・える」
2. 一つ一つ挙げる。列挙する。「理由は種々―・えられる」
3. 数がそれだけのものになる。「蔵書は五万冊を―・える」
4. その中の一つに加える。数に入れる。「候補者の一人に―・えられる」
5. 拍子をとって歌う。
    >「別れの白拍子をぞ―・へける」〈[義経記](https://dictionary.goo.ne.jp/word/%E7%BE%A9%E7%B5%8C%E8%A8%98/#jn-51295)・六〉
