# とどこお・る〔とどこほる〕【滞る】

［動ラ五（四）］
1. 物事が順調に運ばない。はかどらない。つかえる。「仕事が―・る」
2. 金を支払うべき期限になっても支払わない。「家賃が三月分―・る」
3. 流れがとまる。停滞する。「交通が―・る」
4. ためらう。ぐずぐずする。
