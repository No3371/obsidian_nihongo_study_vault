# いか・れる

［動ラ下一］《「行かれる」の意から》
1.  人に先を越される。してやられる。「またしてもやつに―・れてしまった」
2.  用をなさなくなる。だめになる。「テレビが―・れる」
3.  頭の働き・考え方などがまともでなくなる。「少し―・れているんじゃないか」「―・れた服装」
4.  心をうばわれる。夢中になる。「一目で彼女に―・れてしまった」
