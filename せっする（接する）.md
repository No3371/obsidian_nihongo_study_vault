# せっ・する【接する】

［動サ変］［文］せっ・す［サ変］
1. ひと続きにつながる。また、つないでひと続きにする。「川に―・する住宅地」「上下を―・する」
2. さわる。触れる。「肩が―・する」
3. 応対する。「笑顔で人に―・する」
4. 出会う。出くわす。「名作に―・する」「朗報に―・する」
5. 数学で、曲線や直線が他の曲線や直線などと一点だけで出合う。
6. 近くに寄せる。近づける。「ひざを―・する」「軒を―・する」
