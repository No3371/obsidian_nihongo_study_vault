# えら・ぶ【選ぶ／▽択ぶ】

［動バ五（四）］
1. 多くの中から目的や基準にかなうものを取り出す。選択する。よる。えらむ。「贈り物を―・ぶ」「すぐれた作品を―・ぶ」「フォアボールを―・ぶ」
2. 抜き出してその職に任ずる。選び出す。「議長に―・ばれる」
3. （あとに打消しの語を伴って用いる）区別する。えり好みする。「目的のためには手段を―・ばない」
4. （「撰ぶ」とも書く）著作を集めて書物を作る。撰する。「歌集を―・ぶ」
    

\[可能\]**えらべる**
