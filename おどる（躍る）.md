# おど・る〔をどる〕【躍る】

［動ラ五（四）］

1. 飛び跳ねる。跳ね上がる。「網の中で銀鱗 (ぎんりん) が―・る」
2. 喜びや期待などで鼓動が激しくなる。わくわくする。「希望に心が―・る」「血湧き肉―・る」
3. 激しく揺れ動く。「でこぼこ道で車体が―・る」
4. 書いた文字などが乱れる。踊る。「手紙の字が―・っている」
