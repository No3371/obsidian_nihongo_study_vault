# とり‐あつか・う〔‐あつかふ〕【取（り）扱う】

［動ワ五（ハ四）］

1. 物を動かしたり操作したりする。「機械を―・う」「乱暴に―・う」
2. 物事を処理する。とりはからう。担当する。さばく。「事務を―・う」「輸入品を―・う」
3. 人を世話する。もてなす。接待する。「正客として―・う」「丁重に―・う」
