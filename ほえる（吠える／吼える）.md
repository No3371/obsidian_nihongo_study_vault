# ほ・える【×吠える／×吼える】

［動ア下一］［文］ほ・ゆ［ヤ下二］

1. 獣などが大声で[鳴く](なく（泣く／鳴く／啼く）)。「虎が―・える」
2. 風・波などが、荒れて大きな音を立てる。「荒海が―・える」
3. 大声で話す。わめく。どなる。「壇上で―・える」
4. 雄叫 (おたけ) びをあげる。気合いを入れるために大声を出す。「マウンド上で―・える投手」
