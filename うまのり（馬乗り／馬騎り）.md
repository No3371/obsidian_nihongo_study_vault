# うま‐のり【馬乗り／馬▽騎り】

1. 馬に乗ること。また、乗る人。
2. 馬に乗るような姿勢で人や物にまたがること。「─になって毆りつける」
3. 羽織の背や肌ジュバン・甚兵衛などの脇の裾 (すそ) などで、縫わずに開けておく部分。
4. 乗馬の巧みな人。
    >「城陸奧守泰盛 (じゃうのむつのかみやすもり) は雙なき─なりけり」〈[徒然](https://dictionary.goo.ne.jp/word/%E5%BE%92%E7%84%B6%E8%8D%89/#jn-148773)・一八五〉
