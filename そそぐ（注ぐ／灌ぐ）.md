# そそ・ぐ【注ぐ／×灌ぐ】

［動ガ五（四）］《室町時代ごろまで「そそく」》

1. 
    1. ㋐流れ入る。流れ込む。「淀川は大阪湾に―・ぐ」
    2. ㋑雨や雪などがとぎれなく降りかかる。「雨が―・ぐ」「降り―・ぐ」
2. 
    1. ㋐流し入れる。また、容器に水などをつぐ。「田に水を―・ぐ」「茶碗に酒を―・ぐ」
    2. ㋑涙を[流す](ながす（流す）)。「熱い涙を―・ぐ」
    3. ㋒水などを上からかける。ふりかける。「盆栽に水を―・ぐ」
    4. ㋓もっぱら、その方へ向ける。一つことに集中する。「心血を―・ぐ」「視線を―・ぐ」「全力を―・ぐ」
        

\[可能\]**そそげる**
