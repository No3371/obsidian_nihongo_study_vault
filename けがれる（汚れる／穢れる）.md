# けが・れる【汚れる／×穢れる】

［動ラ下一］［文］けが・る［ラ下二］
1.  清らかさ、純粋さ、神聖さなどが損なわれて、よごれた狀態になる。[よごれる](よごれる（汚れる）)。「耳が─・れる」「神殿が─・れる」
2.  名譽や誇りに傷がつく。「履歴が─・れる」
3.  女性が貞操を失う。
4.  死・出產・月経などにかかわって忌むべき狀態になる。
