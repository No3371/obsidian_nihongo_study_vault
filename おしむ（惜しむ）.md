# おし・む〔をしむ〕【惜しむ】

［動マ五（四）］

1.  心残りに思う。残念がる。「散る花を―・む」「別れを―・む」「人に―・まれて死ぬ」
2.  金品などを出すことを、もったいないと思う。出し惜しむ。「わずかの出費を―・んで大損をする」「骨身を―・まず働く」
3.  大切に思う。尊重する。「寸暇を―・む」「名を―・む」
4.  （「愛しむ」とも書く）愛する。めでる。慈しむ。
