# なか‐ごろ【中頃】

1.[ 中ほど](なかほど（中程）)の場所。中ほどの時期。物事の中間。途中。「坂の中頃にあるポスト」「番組の中頃にコマーシャルが入る」「五月の中頃」
2. 歴史上、あまり遠くない昔。中ほどの時代。
