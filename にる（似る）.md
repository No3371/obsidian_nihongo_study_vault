# にる【似る】

［動ナ上一］［文］［ナ上一］

1. 物の形が見た目に同じようである。「双子のように**に**ている」「夕顔に**に**た白い花」
2. 性質・状態などがほとんど同じである。「意味の**に**た言葉」「親に**に**てそそっかしい」
3. （「…に似ず…だ」の形で）それから受ける印象のとおりであろうと判断されるのとは実際には違って…である。「外見に**に**ずたくましい」
4. それに相応すると判断される。…ようだ。
