# は・く【×穿く／履く／×佩く／▽帯く】

![[@1.gif]]［動カ五（四）］

1. （穿く）衣服などを、足先から通して下半身につける。「ズボンを―・く」「袴 (はかま) を―・く」
2. （履く）履物を足につける。「靴を―・く」「たびを―・く」
3. （佩く・帯く）刀・矢などを腰につける。身に帯びる。「太刀を―・く」
    

\[可能\]**はける**
