# うまれ【生（ま）れ】

1.  生まれること。出生 (しゅっしょう) 。誕生。「昭和の生まれ」
2.  生まれた所。出生地。「生まれは東京」
3.  その人の生まれた家柄。素性。「醫者の家の生まれ」
4.  生まれたときからの性質。天性。
