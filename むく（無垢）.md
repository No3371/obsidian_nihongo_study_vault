# む‐く【無×垢】

［名・形動］
1.  仏語。煩悩のけがれを離れて、清浄であること。
2.  けがれがなく純真なこと。うぶなこと。また、そのさま。「無垢な少女」
3.  金・銀などがまじりけのないこと。
4.  和服で、表裏同色の無地の共布で仕立てた長着。白無垢は婚礼衣装に、色無垢は略礼装に用いられる。
