# いっ‐しょ【一緒】

《「一所」から》
1. 一つにまとめること。ひとまとめ。「荷物を一緒にする」「人の分も一緒に扱う」
2. 区別のないこと。同一。「一緒の着物」「意見が一緒になる」
3. ともに同じことをすること。「一緒に遊ぶ」
4. （ふつう「御一緒する」の形で用いて）「連れ立つ」「ともに行く」を、同行する相手を敬っていう語。「そこまで御一緒させてください」
5. 同時に行われること。「一緒に買った物」「着いたのは一緒だ」
