# くず・れる〔くづれる〕【崩れる】

［動ラ下一］［文］くづ・る［ラ下二］
1.  まとまった形をし、安定していたものが、[支える](ささえる（支える）)力を失ってこわれる。また、こわれてばらばらになる。「がけが―・れる」「積み荷が―・れる」
2.      
    1.  ㋐整っていた形や状態が乱れる。「人垣が―・れる」「隊形が―・れる」        
    2.  ㋑整った、好ましい形や状態が悪くなる。「姿勢が―・れる」「生活態度が―・れる」「信頼関係が―・れる」
3. 天気が悪くなる。「天候は午後からしだいに―・れるでしょう」
4. 同額の小銭に替えられる。「一万円札が―・れますか」
5. 相場などが急落する。「値が―・れる」
