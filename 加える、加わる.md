# 【加える】 と 【加わる】 はどう違いますか？


[加える](くわえる（加える）)　add  
[加わる](くわわる（加わる）)　join  
  
e.g.  
【加える・くわえる】  
鍋に塩を加える（なべに　しおを　くわえる）  
add some salt into the pod  
  
彼を友達リストに加える  
add him to my friends list  
  
【加わる・くわわる】  
その活動に加わる（その　かつどうに　くわわる）  
join in the activity  
  
彼の友達リストに加わる  
I joined in his friends list.
