# ひき‐つ・ける【引（き）付ける】

［動カ下一］［文］ひきつ・く［カ下二］

1.  近くに引き寄せる。手元に近づかせる。「明かりで蟲を─・ける」
2.  （「惹き付ける」とも書く）人の心を誘い寄せる。「人を─・ける魅力」「巧みな話術に─・けられる」
3.  無理やり他の場合にあてはめる。こじつける。「自分にいい方に─・けて考える」
4.  痙攣 (けいれん) を起こす。特に、小児が発作性の全身痙攣を起こす。「赤ん坊が夜中に─・ける」
