# り‐がる【尻軽】

［名・形動］
1. 動作の活発なこと。また、そのさま。
2. 落ち着きがなく、行動の軽々しいこと。また、そのさま。「尻軽な振る舞いは慎め」
3. 女の浮気なこと。また、そのさま。
