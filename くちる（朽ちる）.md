# く・ちる【朽ちる】

［動タ上一］［文］く・つ［タ上二］
1.  腐って形がくずれたりぼろぼろになったりする。「―・ちて今にも壊れそうな廃屋」
2.  評判が衰えてしまう。すたれる。「今なお―・ちることのない名声」
3.  むなしく人生を終える。「世に出ることもなく―・ちる」
