# くたび・れる【草＝臥れる】

［動ラ下一］［文］くたび・る［ラ下二］
1. 長時間からだや頭を使ったため、疲れて元気がなくなる。「歩きつづけて─・れる」
2. 年老いたり苦労が続いたりして気力や若さを失う。「人生に─・れる」「生活に─・れた顔」
3. 衣服などが長く使われたため、古く[みすぼらし**く**](みすぼらしい)なる。「─・れた背広」
4. （他の動詞の連用形に付いて）その動作を続けるのが、疲れていやになる。「待ち─・れる」
    →[疲 (つか) れる](つかれる（疲れる）)［用法］
        

#### 類語

-   [疲れる(つかれる)](つかれる（疲れる）)
