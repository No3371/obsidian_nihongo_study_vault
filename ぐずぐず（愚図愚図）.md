# ぐず‐ぐず〔ぐづぐづ〕【愚図愚図】

![[@1.gif]]［副］(スル)

1. のろのろといたずらに時間を費やすさま。「借金の返済を愚図愚図（と）引き延ばす」
2. ぶつぶつ不平を言うさま。また、幼児が[むずかる](むずかる（憤る）)さま。「いつまでも愚図愚図言うな」
3. 天候がはっきりしないさま。「梅雨時の愚図愚図した天気」
4. 物の煮えたつ音。また、そのさまを表す語。
