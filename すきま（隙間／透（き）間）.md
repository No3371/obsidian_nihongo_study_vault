# すき‐ま【隙間／透（き）間】

1.  物と物との間の、わずかにあいている所。「―だらけの板塀」
2.  （比喩的に）普通には気づきにくいところ。盲点。「法の―を衝く」「―産業」
3.  あいている時間。ひま。「家事の―をみて勉強する」
4.  わずかな気のゆるみや油断。「心の―につけ込む」
