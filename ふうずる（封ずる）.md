# ふう・ずる【封ずる】
［動サ変］［文］ふう・ず［サ変］
1.  [封](ふう（封）)をする。「小包をしっかり─・ずる」
2.  出入り口などを閉じてふさぐ。「空港を─・ずる」「敵の退路を─・ずる」
3.  自由な発言・行動ができないようにする。「口を─・ずる」「反対派の動きを─・ずる」
4.  神仏の通力などによって閉じ込める。「お札で蟲を─・ずる」
