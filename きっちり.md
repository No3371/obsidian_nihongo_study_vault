# きっちり

［副］(スル)
1. [すきま](すきま（隙間／透き間）)や[ずれ](ずれ)がないさま。ぴったり。「ふたをきっちり（と）閉める」「答えがきっちり（と）合う」
2. 時間や数量に[端数](はすう（端数）)がないさま。きっかり。「―三時に着く」「一万円―」
3. 物事を確実に行うさま。「毎日―（と）予習をする」
4. よく整っていて[乱れ](みだれ（乱れ／紊れ）)のないさま。
