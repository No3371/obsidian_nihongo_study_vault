# めぐら・す【巡らす／▽回らす／×廻らす】

［動サ五（四）］
1. 周りを囲ませる。「垣根を家の周りに―・す」
2. 回転させる。まわす。「きびすを―・す」「頭 (こうべ) を―・す」
3. あれこれと心を働かせる。「思いを―・す」「はかりごとを―・す」
4. ふれまわって知らせる。回状で知らせる。    
    >「堂の飾り、仏の御具など、―・し仰せらる」〈[源](https://dictionary.goo.ne.jp/word/%E6%BA%90%E6%B0%8F%E7%89%A9%E8%AA%9E/#jn-69890)・松風〉
