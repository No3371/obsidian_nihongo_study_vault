# たい・する【対する】

［動サ変］［文］たい・す［サ変］
1. 二つのものが向かい合う。また、あるものに向き合う。「机を挟んで二人が―・する」「海に―・して建つ家」
2. ある対象に向かう。「家族に―・する思いやり」「将来に―・する期待」
3. 人に接する。応対する。「客に愛想よく―・する」
4. 立ち向かう。相手にして争う。「最強チームに―・する」
5. 対応する。応じる。「質問に―・する答え」「記事に―・して反響があった」
6. 比較する。比べる。「プロに―・しても見劣りしない」
7. 対 (つい) になる。「善に―・する悪」
