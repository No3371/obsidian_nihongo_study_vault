# ばら・す

［動サ五（四）］
1. 解体してばらばらにする。「機械を―・して修理する」「肉を―・す」
2. 殺す。「裏切り者を―・す」    
\[補説\]「殺す」と書いて「ばらす」と読ませることもある。
3. 秘密などを人に知らせる。暴露する。あばく。「内紛を―・す」
4. 不法に売り払う。また、売りとばす。「盗品を―・す」
5. 針にかかった魚を釣り上げる途中で取り逃がす。「大物を―・してくやしがる」
    

\[可能\]**ばらせる**
