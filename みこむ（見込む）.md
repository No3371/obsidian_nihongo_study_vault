# み‐こ・む【見込む】

［動マ五（四）］
1. あてにする。望みをかける。「値上がりを―・んで大量に仕入れる」「君を男と―・んで頼む」
2. 将来有望だと思う。「―・まれて養子になる」
3. 予想して勘定に入れる。「損失を―・む」
4. ねらいをつけたものに執念深くとりつく。「蛇に―・まれたカエル」
