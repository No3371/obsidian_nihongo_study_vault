# しず‐か〔しづ‐〕【靜か／▽閒か】

［形動］［文］［ナリ］
1. 耳ざわりな物音や聲がしないさま。ひっそりしているさま。「─な宿」「蟲の聲が聞こえる─な夜」
2. 目につく動きがなく、ゆったりとしているさま。慌ただしくないさま。「─な川の流れ」「─に椅子を引いて座る」
3. 落ち著いているさま。気持ちに動揺や亂れがないさま。穏やか。「─な境地」「─に餘生を送る」
4. 性格がおとなしく、口數が少ないさま。「─な人柄」
    

\[派生\]**しずかさ**［名］
