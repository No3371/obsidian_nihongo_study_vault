# きゅう‐くつ【窮屈】

［名・形動］
1.  空間や場所にゆとりがなく、自由に動きがとれないこと。また、そのさま。「窮屈な服」「座席が窮屈になる」
2.  思うようにふるまえず気詰まりであること。また、そのさま。「お偉方ばかりで窮屈な会だ」「窮屈な思いをする」
3.  形式張って堅苦しいこと。融通のきかないこと。また、そのさま。「窮屈で面白みのない男」「窮屈に考えすぎる」
4.  物や金が不足してままならないさま。「窮屈な暮らし」
    

\[派生\]**きゅうくつがる**［動ラ五］**きゅうくつさ**［名］
