# まくら【枕】

1. 寝るときに頭をのせる寝具。「枕が変わると眠れない」「氷枕」「ひざ枕」
2. 寝ている頭の方。また、頭のある方角。「東を枕に寝る」
3. 寝ること。宿ること。「旅枕」
4. 長い物を横たえるとき、下に置いてその支えとするもの。「枕木
5. 物事のたね。よりどころ。「歌枕」
6. 話の前置き。落語などで、本題に入る前の短い話。「時局風刺を枕に振る」
7. 地歌・箏曲 (そうきょく) で、手事 (てごと) の導入部分。また、義太夫節で、一段の導入部分。
