# み‐とお・す〔‐とほす〕【見通す／見▽透す】

［動サ五（四）］
1. 初めから終わりまで休まずに見る。見つづける。「退屈な芝居を最後まで―・す」
2. 何にもさえぎられずに遠くまで一目に見る。「窓から富士山を―・す」
3. 人の心や目に見えない内面の物事を見抜く。「彼の意図を―・す」
4.  物事のなりゆきや、将来のことを予測する。「景気の動向を―・す」
