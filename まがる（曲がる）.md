# まが・る【曲（が）る】

［動ラ五（四）］
1. まっすぐなものが弓形・くの字形などになる。「雪の重みで竹が―・る」「背中が―・る」
2. 道の方向が変わる。「土手の手前で道が―・っている」
3. 進行する向きを変える。「次の角を左に―・る」
4. 正しい位置や方向からそれる。傾く。ゆがむ。「ネクタイが―・っている」
5. （「まがった」「まがっている」の形で）性質や考えなどが正しくない。すなおでない。「心の―・っている人」「―・ったことはきらいだ」
6. 衰える。「身代が―・る」
