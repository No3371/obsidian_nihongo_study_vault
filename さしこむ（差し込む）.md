# さし‐こ・む【差（し）込む】

［動マ五（四）］
1. 物の中やすきまなどに、他の物をさしたりはさんだりするようにして入れる。差し入れる。「プラグをコンセントに―・む」「本の間に手紙を―・む」
2. （「射し込む」とも書く）光が入ってくる。「部屋に朝日が―・む」
3. 胸・腹が急に激しく痛む。癪 (しゃく) を起こす。「胃が―・む」
4. 口出しをする。入れ知恵をする。
    >「北の方に―・まれ」〈浄・天鼓〉
        

\[可能\]**さしこめる**
