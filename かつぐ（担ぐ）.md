# かつ・ぐ【担ぐ】

［動ガ五（四）］

1. 物を持ち上げて肩にのせ支える。[になう](になう（担う／荷う）)。「荷物を―・ぐ」「鉄砲を―・ぐ」
2. 自分たちの組織や集団の代表者の地位に据えて押し立てる。祭り上げる。「最年長者を会長に―・ぐ」
3. からかって人をだます。一杯食わせる。「友人にうまく―・がれた」
4. 縁起を気にする。迷信にとらわれる。「―・ぐ性分」「験 (げん) を―・ぐ」
    

\[可能\]**かつげる**
