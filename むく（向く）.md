# む・く【向く】

**\#1**［動カ五（四）］
1. その方向に正面が位置するようにする。[対する](たいする（対する）)。面する。「上を―・く」「東に―・く」
2. その方向を指し示す。「磁石の針は北を―・く」
3. その方向・状態にかたむく。「気が―・く」「心は故郷に―・いている」
4. 適する。[ふさわしい](ふさわしい（相応しい）)。「若者に―・いた仕事」
