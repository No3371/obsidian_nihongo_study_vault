# へだた・る【隔たる】

［動ラ五（四）］
1. 間に距離があって離れる。また、距離ができて離れる。遠ざかる。「約10キロ―・っている」
2. 間に物があって遮られる。「両県は山脈で―・る」
3. 時間的に離れる。また、年月が過ぎる。月日がたつ。「今を―・る百年の昔」
4. 物事の間に開き、違いがある。「双方の実力はかなり―・っている」
5. 関係がうとくなる。疎遠になる。「心が―・る」
